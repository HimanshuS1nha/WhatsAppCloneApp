## Features

- Realtime chat using socket.io
- Audio and video calling through webrtc
- Online and offline status
- Login/Logout

## TECH STACK USED

- React Native
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- Prisma
- WebRTC
